# 📊 Sales Executive Dashboard (Power BI)

## 📌 Project Overview

This project presents a **Sales Executive Dashboard** built using **Power BI** to analyze sales performance across products, regions, and sales managers.

The dashboard provides key business insights such as total sales, profit, transactions, and customer behavior, helping stakeholders make data-driven decisions.

---

## 🛠️ Tools & Technologies Used

* Power BI
* Excel (Data Source)
* Data Cleaning & Transformation (Power Query)
* Data Modeling
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

* File: `Sales Raw Data.xlsx`
* Contains:

  * Product details
  * Sales transactions
  * Quantity sold
  * Sales amount
  * Profit
  * Country & Date

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Quantity Sold
* Total Transactions
* Total Sales
* Average Basket Value
* Average Order Quantity
* Total Profit

---

### 🔹 Visualizations

* 📈 Transactions by Date (Time Series Analysis)
* 🍩 Transactions by Product (Donut Chart)
* 📊 Transactions by Sales Manager (Bar Chart)
* 🌍 Transactions by Country (Map Visualization)

---

### 🔹 Filters (Slicers)

* Product
* Sales Manager
* Country
* Date Range

---

## 🔍 Key Insights

* Identified top-performing products and regions
* Analyzed sales trends over time
* Compared performance of sales managers
* Evaluated customer purchase behavior

---

## 🚀 How to Use

1. Download the `.pbix` file
